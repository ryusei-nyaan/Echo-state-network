# Echo-state-network
リザバーコンピューティングの１つです．

###### Julia version = 1.5.1 

#### fit関数

時系列の学習を行います．

##### x:学習用時系列
##### n:リサバー状態ベクトルの次元
##### τ:正則化パラメータ，正の値
##### pwij:i番目のリザバーノードからj番目への重み行列の0以外の要素の比率
##### pwin:入力の重み行列の0以外の要素の比率
##### pwself:1時刻前の出力フィードバックの重み行列の0以外の要素の比率
##### pwb:バイアス項の重み行列の0以外の要素の比率
##### S:最適化回数
